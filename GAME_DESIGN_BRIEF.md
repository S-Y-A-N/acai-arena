# Game Design Brief

## Core idea
ACAI Sovereign Arena turns the academic project into a live competition. Visitors are not only watching a poster; they are making real AI deployment decisions.

## Why each mode exists
- Model Compression Lab: explains quantization, pruning, confidence filtering, size reduction, accuracy retention.
- RAG Evidence Sprint: explains retrieval and citation-backed answers.
- AI Quality Challenge: explains hallucination, answer evaluation, and academic trust.
- Governance Council: explains privacy, access control, and institutional responsibility.
- Agent Command: explains ACAI's multi-agent architecture.

## Audience fit
- Professors/judges: use Grand Tournament.
- Business visitors: Governance + Quality Challenge.
- IT/technical visitors: Compression + RAG.
- Children/general visitors: Agent Command.
