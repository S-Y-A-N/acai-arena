# ACAI Sovereign Arena — Million-Dollar Exhibition Game

This is the upgraded executive game package for the ACAI senior exhibition.

## What is different in this version

- Luxury cinematic UI with animated particle background.
- Full Arabic/English toggle across the interface.
- 1–4 player competition.
- Grand Tournament mode or Single Elite Mode.
- Live leaderboard, timer, turn system, boosters, and final ceremony.
- Research-based game modes:
  1. Model Compression Lab
  2. RAG Evidence Sprint
  3. AI Quality Challenge
  4. Governance Council
  5. Agent Command
- Research Core page explaining why each game matters.
- Print Studio page and printable templates.

## How to run

1. Extract the ZIP.
2. Open `index.html` in Chrome or Edge.
3. Press F11 for full-screen exhibition mode.

## Recommended booth setup

- Use a large touchscreen or iPad/laptop.
- Put printed challenge cards next to the device.
- When visitors finish, give them the matching agent card/sticker/chocolate.
- Use the Print Studio templates for table signs and agent cards.
