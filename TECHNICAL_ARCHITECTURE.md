# Technical Architecture

This is an offline browser game using HTML, CSS, and vanilla JavaScript.

Files:
- index.html: main game application
- style.css: luxury UI system
- app.js: game state, scoring, multilingual rendering, and modes
- print-templates.html: printable assets
- assets/: ACAI logo and six agent images

Future extension:
- Host it online and link QR to index.html.
- Add real ACAI API calls to evaluate live answers.
- Add persistent leaderboard with Firebase/Supabase or local server.
- Add QR result cards per player.
