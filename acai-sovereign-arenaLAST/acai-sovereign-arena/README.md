# ACAI Sovereign Arena — Executive Exhibition Experience

The polished, final tournament game for the ACAI senior exhibition. Bilingual
Arabic / English, fully offline, built for professors, executives, judges,
students, and visiting families.

## What's in this version

- Refined dark "sovereign" theme with gold accents and holographic motion.
- Full Arabic / English toggle across the whole interface (right-to-left in Arabic).
- 1–4 player competition with a live leaderboard, timer, streak scoring, and boosters.
- Grand Tournament (all five modes) or Single Elite Mode (one mode).
- Five visitor-friendly modes:
  1. Efficient AI Lab — why ACAI runs as a smaller, faster, reliable model.
  2. Evidence Sprint — build the retrieve-then-cite flow in order.
  3. Trust & Verify — spot confident-but-wrong answers and untraceable sources.
  4. Governance Council — privacy, access control, and accountability.
  5. Agent Command — match each request to the right ACAI agent.
- Research Core page that explains the academic value for judges.
- A final ceremony with a podium and an ACAI recognition card per champion.

## Questions

Every question is written to be correct and understandable for a general
visitor. Deep, internal, report-style detail has been removed so that families,
students, and non-technical guests can play and learn without being lost.

## How to run

1. Keep all files together (index.html, style.css, app.js, and the assets folder).
2. Open `index.html` in Chrome or Edge.
3. Press F11 for full-screen exhibition mode.

## Fonts & offline

The interface uses IBM Plex Sans + IBM Plex Sans Arabic, bundled locally in
`assets/fonts`, so the typography looks identical with no internet connection.
If any agent image is missing, a built-in fallback renders in its place.

## Recommended booth setup

- Use a large touchscreen, iPad, or laptop on the stand.
- Professors / judges: Grand Tournament.
- Business guests: Governance + Trust & Verify.
- Technical guests: Efficient AI Lab + Evidence Sprint.
- Families / general visitors: Agent Command.
- When a visitor finishes, present them with the ACAI recognition card and the
  matching agent card shown on the result screen.
